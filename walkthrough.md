# UCL Website Walkthrough

I have successfully created the Urian Cybersecurity League website with the "Blue Ocean" theme! 🌊🛡️

## What's Included
- **Tech Stack**: React 18, Vite, TypeScript, Tailwind CSS v3.
- **Theme**: Custom "Blue Ocean" palette (#4FA3FF, #A7E0FF, #FFD6A5) with friendly, rounded aesthetics.
- **Components**:
  - `Navbar`: Responsive with smooth scroll and mobile menu.
  - `Hero`: Features a mascot placeholder and animations.
  - `MissionCard` & `FeatureCard`: Reusable cards with hover effects.
  - `Footer`: Clean and simple.
- **Content**: All text content (About, Vision, Missions) has been implemented as requested.

## Verification Results
### Build
The project builds successfully with strict type checking:
```bash
npm run build
```
Output: `dist/` folder ready for Vercel deployment.

## How to Run
1. **Development**:
   ```bash
   npm run dev
   ```
2. **Preview Production Build**:
   ```bash
   npm run build
   npm run preview
   ```
