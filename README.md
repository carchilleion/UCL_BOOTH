# Portfolio Website

A simple and clean portfolio website built with React.

## Features

- Responsive design
- Modern UI with smooth transitions
- Sections: About, Projects, Contact
- Easy to customize

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Customization

1. **Update your information:**
   - Edit `src/components/Header.jsx` to change your name and title
   - Edit `src/components/About.jsx` to update your bio and skills
   - Edit `src/components/Projects.jsx` to add your projects
   - Edit `src/components/Contact.jsx` to update contact information

2. **Change colors:**
   - Edit the color values in `src/styles/App.css`

3. **Add more sections:**
   - Create new components in `src/components/`
   - Import and add them to `src/App.jsx`

## Tech Stack

- React 18
- Vite (build tool)
- CSS3
